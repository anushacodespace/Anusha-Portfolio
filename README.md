# Anusha Dev — Portfolio

**React / Next.js developer — Frontend + Basic API integration**

This is my personal portfolio built with Next.js and Material UI. It includes a small user list with a dynamic route demo, responsive UI, light/dark theme toggle, and a contact page (links to email / LinkedIn).

**Live demo:** https://anusha-portfolio-three.vercel.app/  
**Source:** https://github.com/anushacodespace/Anusha-Portfolio

---

## What’s in this repo

- `app/` — Next.js app router pages and components (Home, About, Users, Contact)
- `public/` — static assets (images, icons)
- `backend/` — optional Express server used locally for demo data (not required on Vercel)
- `app/data/users.js` — mock user data used when no backend available
- `.gitignore` — excludes node_modules, build artifacts, env files

---

## Tech stack

- Next.js (App Router)
- React
- Material-UI (MUI)
- Optional: Express (for local mock API)
- Hosted on Vercel
